bindQuery [ beta ]
bindQuery is a Jquery plugin which provides a feature of Angular, this feature is ng-repeat that we use to bind our Json data to HTML.

Generate Json data:

Rank Country Population
$Rank$ $Country$ $Population$

Generate Json data order by Country ascending :

Rank Country Population
$Rank$ $Country$ $Population$

Generate Json data order by Rank descending :

Rank Country Population
$Rank$ $Country$ $Population$
